# CompBio Edu

Computational Biology Educational Web Platform.

## Features

- Protein Prism — 3D protein structure viewer
- Genomic Navigator — BLAST search, evolutionary timeline, and glossary

## Technologies

- React
- Vite
- TypeScript
- Tailwind CSS
- 3Dmol.js

## Local Development

```bash
npm install && npm run dev
```
